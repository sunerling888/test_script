
# 单接口+多接口

- 测试框架
    - nosetests + nose-html-reporting

- 单接口(脚本)
    - 测试数据,存放到csv文件
    - 每个测试接口,对应多条测试case

- 执行
- 生成测试报告

- 多个关联接口(脚本)   -----[卖家/买家/游客]
    - 测试数据,存放到csv文件
    - 封装会话、csv
    - 设置从登录开始到退出登录结束
        - 例子: 登录 -> 个人中心接口(post) -> 设置页面(get) -> 退出登录


- 单个测试流程执行
- 多个测试流程执行
- 生成测试报告
